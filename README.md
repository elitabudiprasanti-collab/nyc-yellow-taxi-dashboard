# 🚖 NYC Yellow Taxi Dashboard

## 📌 Project Overview

This project presents an advanced spatial and statistical analysis of NYC Yellow Taxi Trip Data using interactive dashboard visualization.

The dashboard was developed using Streamlit and includes:

- Exploratory Data Analysis (EDA)
- Spatial Visualization
- Density Mapping
- Clustering Analysis
- Statistical Correlation Analysis
- Interactive Dashboard Features

---

# 📊 Methods Used

## Data Preprocessing
- Missing value handling
- Outlier removal
- Geographic filtering
- Feature engineering

## Clustering Methods
- HDBSCAN
- KMeans
- DBSCAN
- K-Medoids (PAM)

## Evaluation Metrics
- Silhouette Score
- Davies-Bouldin Index
- Noise Percentage

---

# 🗺️ Dashboard Features

- Dataset Overview
- Correlation Heatmap
- Pickup Density Map
- Spatial Clustering Map
- Peak Hour Analysis
- Statistical Insight
